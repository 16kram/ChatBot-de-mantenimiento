# Chatbot de Mantenimiento ⚙️

Este proyecto es un chatbot basado en JavaScript que utiliza la API de OpenAI para responder a preguntas relacionadas con mantenimiento eléctrico. La interfaz es simple y efectiva, diseñada para ofrecer respuestas rápidas y relevantes en un entorno de chat similar a aplicaciones de mensajería. Ideal para soporte técnico o asistencia rápida en consultas sobre mantenimiento.

### Características principales:
- **Interfaz de chat**: La interfaz de usuario presenta un cuadro de texto para ingresar preguntas y un área de visualización para respuestas.
- **Conexión a la API de OpenAI**: Envia consultas de los usuarios a la API de OpenAI y devuelve respuestas generadas por ChatGPT en tiempo real.
- **Desplazamiento automático**: La caja de chat se desplaza automáticamente al final con cada nuevo mensaje para mejorar la experiencia del usuario.
- **Estilo personalizable**: Estilos CSS básicos que puedes personalizar para adaptarlo a la estética de tu proyecto.
  
### Requisitos:
- **API Key de OpenAI**: Necesitarás una clave de API de OpenAI para autenticar las solicitudes.
- **Servidor local**: Puedes utilizar un servidor local como XAMPP o cualquier servidor HTTP simple para ejecutar el proyecto.

### Archivos clave:
- **`index.html`**: Contiene la estructura básica del chatbot.
- **`main.js`**: El script JavaScript que controla la interacción con la API y la lógica de despliegue de mensajes.
- **`style.css`**: Contiene los estilos para la caja de chat y los mensajes.

### Cómo usar el proyecto:
1. Clona el repositorio en tu máquina local.
2. Configura tu clave API de OpenAI en el archivo `main.js`.
3. Ejecuta el proyecto en un servidor local.
4. Interactúa con el chatbot enviando preguntas sobre mantenimiento eléctrico y recibe respuestas en tiempo real.
